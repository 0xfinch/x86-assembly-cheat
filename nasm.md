# NASM

Netwide assembler. TODO what does Netwide mean?

One of the most popular for Linux.

Open source.

Intel like syntax.

Input formats:

- `.asm`, Intel like syntax

Most useful command line options:

- `-w+all`: enable all warnings

## f

Output format:

    nasm -f elf32 main.asm

List all formats:

    nasm -hf
